Modal

Guide for modal creation: (according to webinar Модуль 5. Вебинар №9. Позиционированные элементы)

  1. Create button to open modal
  2. Create <div class=backdrop> </div>
    • .backdrop { position: fixed; top: 0; left: 0; width: 100%; height: 100%; background-color: rgba(255, 165, 0, 0.5) }
  3. Add <div class="modal"> </div> - means modal content, like <div class="backdrop"> <div class="modal"> </div> </div>
    • .modal { position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%); width: 600px; height: 400px; background-color: white; }
  4. Add necessary content ot modal
  5. Add close button
  6. Add .backdrop.is-hidden (or .backdrop--is-hidden if we use BEM) class for div with backdrop, like
    • .backdrop--is-hidden { opacity: 0; pointer-events: none; }
  7. Add JS script:
    • const refs = { openModalBtn: document.querySelector('[data-open-modal]'), closeModalBtn: document.querySelector('[data-close-modal]'), backdrop: document.querySelector('[data-backdrop]'), };

      refs.openModalBtn.addEventListener('click', toggleModal); refs.closeModalBtn.addEventListener('click', toggleModal); refs.backdrop.addEventListener('click', logBackdropClick);

      function toggleModal() { refs.backdrop.classList.toggle('backdrop--is-hidden'); }

      function logBackdropClick() { console.log('This is a click to the backdrop'); }
    • Add data attributes for backdrop, open, close buttons, like
      <button data-open-modal>Open modal </button>

      <div class="backdrop backdrop--is-hidden" data-backdrop > <div class="modal">
      <button data-close-modal>Close modal </button>
      </div> </div>
